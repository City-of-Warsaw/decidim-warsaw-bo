# Decidim::Projects

Component mixing functions of proposals, budgets and accountabilities.

## Usage

Projects will be available as a Component for a Participatory
Space.

## Installation

Add this line to your application's Gemfile:

```ruby
gem "decidim-projects"
```

And then execute:

```bash
bundle
```
